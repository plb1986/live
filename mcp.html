使用说明

工作原理

1. 系统会提取两个列表中每个频道的 tvg-name 属性作为匹配依据

2. 对于列表一中的频道名称，系统会先尝试直接匹配，然后尝试应用映射规则匹配

3. 映射规则：cctv5p → CCTV5+, CGTNALBY → CGTN阿语, CGTNEN → CGTN, cgtnfrench → CGTN法语, cgtndocumentary → CGTN纪录, cgtnru → CGTN俄语, cgtnsp → CGTN西语

4. 4K/8K频道保护：CCTV4K不会匹配CCTV4，CCTV8K不会匹配CCTV8

5. 如果找到匹配项，将列表一中的播放地址（URL行）替换为列表二中的播放地址

6. 结果中只包含匹配成功的频道，未匹配的频道将被排除
